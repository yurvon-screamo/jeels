# Rainbow Uwuwu UI Kit

Набор UI‑компонентов, оформленных в нежной «зефирной» палитре и построенных на TailwindCSS (CDN). Все стили и скрипты лежат в `ui/index.html`.

## Компоненты

- **Typography & Card**: заголовки H1–H3, параграф с описанием и бейдж `Rainbow Tag`. Демонстрирует палитру и тени карточек.
- **Buttons**: основная кнопка с градиентом и эффектом вспышки, мягкая «pearlescent» кнопка, outline‑кнопка, круглые и квадратные иконки‑кнопки с ховером и glow.
- **Inputs & Textarea**: текстовое поле, поле поиска с иконкой, textarea. Подсветка фокуса через мягкие бордеры, ring‑эффект и смену цвета лейблов по фокусу.
- **Toggles & Controls**: градиентный свитч «Rainbow Mode», чекбоксы с кастомным фоном/иконкой, радиокнопки с анимацией скейла внутренней точки.
- **Selects & Dropdowns**: кастомный select с раскрывающимся списком (JS: `toggleDropdown`, `selectOption`), стрелка вращается при открытии; дополнительный hover‑dropdown без JS.
- **Rainbow Composition (Form)**: карточка с градиентным фоном, блюрами и формой (имя, фамилия, e‑mail) и кнопкой «Get Early Access» в обратной цветовой схеме.
